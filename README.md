# Isaac Andress Martinez Graduation Site

Static invitation site for `izzysgraduation.com`.

## RSVP setup

The site links to the published Google Form in `script.js`. The form collects:

- Name
- Attending: yes/no
- Total number of people attending, including the person RSVPing
- Phone or email
- Optional note to Isaac

Put the full party address in the Google Form confirmation message, not in this
site. The response sheet should keep a total-headcount formula so food planning
can use the attending headcount instead of just RSVP rows.

## Photos

The public web-safe copies live in `assets/photos/`: one hero image plus a
16-photo gallery from the refreshed `IzzysGraduation` Photos album. The public
copies have clean names, are resized for the web, and are stripped of export
metadata.

The red hat reference image was used only as visual direction for the red 90s
style and is not included in the publishable site assets.
